### Switch Attributes

| 参数            | 说明                  | 类型    | 可选值 | 默认值  |
| --------------- | --------------------- | ------- | ------ | ------- |
| value / v-model | 绑定值                | boolean | —      | —       |
| disabled        | 是否禁用              | boolean | —      | false   |
| active-color    | switch 打开时的背景色 | string  | —      | #03c4a1 |
| loading         | 是否加载中            | boolean | —      | false   |

### Switch Events

| 事件名称 | 说明                            | 回调参数   |
| -------- | ------------------------------- | ---------- |
| change   | switch 状态发生变化时的回调函数 | 新状态的值 |

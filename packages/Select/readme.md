### Select Attributes

| 参数            | 说明           | 类型            | 可选值 | 默认值 |
| --------------- | -------------- | --------------- | ------ | ------ |
| value / v-model | 绑定值         | string / number | —      | —      |
| label           | 左上角标题文本 | string          | —      | label  |
| disabled        | 是否禁用       | boolean         | —      | false  |

### Select Events

| 事件名称 | 说明                 | 回调参数     |
| -------- | -------------------- | ------------ |
| change   | 选中值发生变化时触发 | 目前的选中值 |

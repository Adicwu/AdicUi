### CheckboxGroup Attributes

| 参数            | 说明               | 类型          | 可选值 | 默认值 |
| --------------- | ------------------ | ------------- | ------ | ------ |
| value / v-model | 绑定值             | string/number | —      | —      |
| column          | 是否列排，默认横排 | boolean       | —      | false  |
| disabled        | 是否禁用           | boolean       | —      | false  |

### CheckboxGroup Events

| 事件名称 | 说明                   | 回调参数        |
| -------- | ---------------------- | --------------- |
| change   | 绑定值变化时触发的事件 | 选中的 Radio 值 |

### Option Attributes

| 参数            | 说明         | 类型            | 可选值 | 默认值 |
| --------------- | ------------ | --------------- | ------ | ------ |
| value / v-model | 值           | string / number | —      | —      |
| label           | 文本内容     | string          | —      | label  |
| disabled        | 是否禁用此项 | boolean         | —      | false  |

*作为Select选项使用

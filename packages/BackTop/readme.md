### BackTop Attributes

| 参数              | 说明                 | 类型          | 可选值 | 默认值 |
| ----------------- | -------------------- | ------------- | ------ | ------ |
| target            | 监听滚动的节点名称   | string        | —      | —      |
| visibility-height | 滚动到此位置显隐按钮 | number/string | —      | 200    |
| right             | 距离页面右边距离     | number/string | —      | 40     |
| bottom            | 距离页面底部距离     | number/string | —      | 40     |

### BackTop Events

| 事件名 | 说明               | 回调参数 |
| ------ | ------------------ | -------- |
| click  | 点击按钮触发的事件 | 点击事件 |

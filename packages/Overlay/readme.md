### Overlay Attributes

| 参数            | 说明                    | 类型          | 可选值                       | 默认值       |
| --------------- | ----------------------- | ------------- | ---------------------------- | ------------ |
| value / v-model | 绑定值                  | string/number | —                            | —            |
| get-container   | 设置自身的生成节点位置  | string        | —                            | 当前所在位置 |
| inner           | 设置内容分布位置        | boolean       | center/top/bottom/left/right | center       |
| position        | 定位属性，和原生css相同 | string        | css position值               | fixed        |
